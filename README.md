# Python Snake

Dependencies:

* Python 3
* Pygame

To run: 

```bash
    $ python3 main.py
```

Purpose of each file:
| File | Purpose |
|------|---------|
| constants.py | holds constants and pygame variables |
| food.py      | holds Food class |
| functions.py | holds general functions |
| main.py      | holds entry point |
| snake.py     | holds Snake classes |
